# ICMDA Master Data — 버전 이력

- 현재 작업 파일은 프로젝트 루트의 `ICMDA Master Data v{N}.xlsx` (가장 큰 N 이 최신).
- 새로 편집할 때마다 N 이 1 증가하고, **직전 버전 파일은 이 `history/` 폴더로 이동**합니다.
- 즉 루트에는 항상 최신본 1개, `history/` 에는 그 이전 버전들이 쌓입니다.
